Requex - Reducer Query Extensions

Requex - Reducer Query Extensions

This library helps you build fold functions or Redux reducers declaratively. It allows you to specify reducers using queries over an event (or action) stream. You can use the resulting function to create a Redux store.

For more information read the Requex docs.
