    public nextAuxillary: Object source

    This contains the auxillary state built up in a reduction pass. For example, this may include accumulator values for folds that cannot easily be derived from looking at the previous state.

    Example:

    Illustration where previous state isn't the same as the accumulator
    From.events().ofAnyType().select(_ => 1).sum(0).select(x => 2 * x)
